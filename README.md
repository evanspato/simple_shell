0x16. C - Simple Shell

This project is the second project done on ALX Africa, its about using the C programming language, to develop our own shell. 

Description: The shell is a command-line interpreter or shell that provides a command line user interface for Unix-like operating systems. The shell is both an interactive command language and a scripting language, and is used by the operating system to control the execution of the system using shell scripts.

Compilation: You can compile the shell this way:
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hsh

Output examples:
vagrant@vagrant-ubuntu-trusty-64:~/simple_shell$ ./hsh
